# 703. Kth Largest Element in a Stream
> https://leetcode.com/problems/kth-largest-element-in-a-stream/description/

- `Heap / Priority Queue`

```cpp
class KthLargest {
    int k;
    priority_queue<int, vector<int>, greater<int>> pq;
public:
    KthLargest(int k, vector<int>& nums): k(k) {
        for (int x: nums) {
            pq.push(x);
            if (pq.size() > k) pq.pop();
        }
    }
    
    int add(int val) {
        pq.push(val);
        if (pq.size() > k) pq.pop();
        return pq.top();
    }
};
```
