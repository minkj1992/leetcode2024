# 1046. Last Stone Weight
> https://leetcode.com/problems/last-stone-weight/description/

- `Heap / Priority Queue`

```cpp
class Solution {    
public:
    int lastStoneWeight(vector<int>& stones) {
        priority_queue<int> pq(stones.begin(), stones.end());
        while (pq.size() > 1) {
            int a = pq.top();
            pq.pop();            
            int b = pq.top();
            pq.pop();
            pq.push(a==b ? 0: a-b);
        }
        return pq.size() == 1 ? pq.top(): 0;
    }
};
```
